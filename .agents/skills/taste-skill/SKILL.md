# anti-slop design skill

Bans generic AI visual tells. Enforces taste in typography, color, layout, motion.

**Source:** https://github.com/leonxlnx/taste-skill

## Banned
- Em-dash in visible text
- Fraunces / Instrument Serif as defaults
- Purple gradients + dark mesh hero
- Three equal feature cards
- Beige #f5f1ea / brass #b08947 palette
- Glassmorphism as default
- Inter as default
- Pure #000 or #fff
- Centered hero
- Split-header (left headline + right explainer)
- Version labels, section numbers, scroll cues
- Decorative status dots, rotated text, locale strips
- "Elevate", "Seamless", "Next-Gen"
